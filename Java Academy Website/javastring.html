Java Strings

In this chapter we will discuss our first non-primitive data type & the most widely used, the String. This non-primitive data type is used to store text, this text is stored in variable surrounded by double quotes. As a reminder a non-primitive data type is a data type which refers to an object or objects & non primitive data types are created by the programmer with the String being the only exception to this. Non primitive methods can be manipulated by methods also known as functions, we will discuss methods at a later time.

String example

public class MyNewClass {
public static void main(String[] args) {
String hello = "Greetings";
System.out.println(hello);
}
}

String Methods

The first encounter you will have with methods will be here, since a String is an object it has certain functions or methods that can be used to manipulate the String. Examples of each of these methods are located in the compiler below.

  • The Length method in String will return the length of a String object.
  • Two similar methods for a String are toUpperCase and toLowerCase which will return a string in all Uppercase or all Lowercase.
  • The indexOf method will return the index (Position) of the first occurence of a designated text in a string object.
  • String concatenation is very important and useful, it uses the (+) operator to join two string objects together which is known as concatenation.
  • Special characters will generate errors within a string, we will show you how to use special character such as quotes (") inside a string.
