Java If statements & Conditions

The area of If statements and conditions is a major area in Java, in this area we will go over conditions and if else statements with examples to visually show you how this area functions in relation to Java.

Conditional Statements

As stated in the previous chapters conditional statements are commonly used with boolean data types below are all the conditional statements that can be found in Java.

  • if is used to specify a block of code that is to be executed if a specific condition is true.
  • else is used to specify a block of code that is to be executed if a specific condition is false.
  • if else is used to test a new condition, if the first specific condition is false.

If Statements

if statement syntax, if the condition is true that will execute some code. Note (if) is in lowercase, in uppercase e.g. (If or IF) will create an error.

if (condition) {
// Block of code to be executed if the condition is true
}

Else Statements

else statement syntax, if the condition is false that will execute some code.

if (condition) {
// Block of code to be executed if the condition is true
} else {
// Block of code to be executed if the condition is false
}

Else if Statements

else if statement syntax, if the first condition is false a new condition is specified using the else if statement.

if (conditionOne) {
// Block of code to be executed if the conditionOne is true
} else if (conditionTwo) {
// Block of code to be executed if the conditionOne is false and conditionTwo is true
} else {
// Block of code to be executed if the conditionOne is false and conditionTwo is false
}
