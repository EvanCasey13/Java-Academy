Getting Started

This page focuses on the beginning of your journey in learning basic java, on the left you can see some code which when executed will print "This is my new class" into the console as an output. Every java file must have a class which will contain any code you write, in this case the class is MyNewClass. Each class must begin with a capital letter & the name of your java file must match the name of your class eg public class MyNewClass & MyNewClass.java .

The main method is a method that is present in every java program, it allows for any code inside it to be executed, you don't need to understand any of the keywords before and after main, you'll learn about that in subsequent sections. The most important thing to remember from this is that every java program has a class name which is the same as its file name and that the main method must be in every java program. Inside the main method we use the println() method to print a line to the screen.
