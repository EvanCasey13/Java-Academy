Java Commenting

Java commenting is a important area of programming in this language, comments are generally used to explain what a certain block of code will do. There are two widely used versions of commenting, these are single line comments & multi-line comments. Single line comments start with two forward slashes eg (//), any text between these two forward slashes and the end of the line is ignored. Multi-line comments start with a forward slash and a asterisk (/*) & end with a asterisk and forward slash (*/) with any text in between these being ignored.

Java Comments

This page focuses on Java Commenting a relatively easy concept to learn & understand in Java.

A common rule to use when deciding whether or not to comment something is if another programmer will be using some code you have written hence, commenting what certain things do would be beneficial to others using your code. Comments can also be used if you write some complex code that does many things and yoy want to explain each section of the code so if you come back at a later date you can understand how the code will function.
