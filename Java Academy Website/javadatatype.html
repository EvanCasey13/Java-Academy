Java Data Types

As stated in the previous chapter, Java variables must have a specified type. Data types in Java are divided into two types :

  • Primitive data types which include int, float, double, boolean, char & byte (Note there are more primitive data types, these are not necessary for the scope of this site).
  • Non-Primitive data types which include Strings, arrays & classes (This will be covered in later chapters).

Primitive Data Types

A primitive data type specifies the size and type of variable values, with no further methods, in total there are eight primitive data types in Java. We wil be covering six of these.

- Byte is a data type with a size of 1 bit that stores whole numbers in the range -128 to 127.
- Int is a data type which has a size of 4 bytes that can store whole numbers.
- float is a data type which has a size of 4 bytes that can store fractional numbers up to 7 digits.
- char is a data type which has a size of 2 bytes that can store a single character.
- double is a data type which has a size of 8 bytes that can store fractional numbers up to 15 digits.
- boolean is a data type which has a size of 1 bit that stores either a true or false value.

Numbers

Primitive number types are divided again into two categories :
- Integer types which store whole numbers that can be postive or negative, these include int & byte in our case.
- Float point types which store numbers with a fractional element e.g. one or more decimals, there are two types of this, float & double.

Byte

As stated previously, the byte data type can store numbers from -128 to 127 which can be used instead of int if you know the range of values within the range of values of a byte data type.

public class MyNewClass {
public static void main(String[] args) {
byte myByte = 100;
System.out.println(myByte);
}
}

Int

Int is the preferred data type when we want to store a numeric value, it stores whole numeric values in a very large number range resulting in this data type being more versatile than byte.

public class MyNewClass {
public static void main(String[] args) {
int myInt = 10000;
System.out.println(myInt);
}
}

Floating point types

Should I use float or double? The accuracy of a floating point value indicates how many digits it can have after a decimal point, since double can have 15 digits after a decimal point it is the preferred data value for floating point calculations.

Double

The double data type stores fractional numbers, note you should end a double value with "d" to differentiate it from a float value.

public class MyNewClass {
public static void main(String[] args) {
double myDouble = 27.842345d;
System.out.println(myDouble);
}
}

Float

The float data type stores fractional numbers, note you should end a float value with "f" to differentiate it from a float value.

public class MyNewClass {
public static void main(String[] args) {
float myFloat = 27.843f;
System.out.println(myFloat);
}
}

Booleans

The boolean data type store a value that can only be true or false.

public class MyNewClass {
public static void main(String[] args) {
boolean myBoolean1 = true;
System.out.println(myBoolean1);
boolean myBoolean2 = false;
System.out.println(myBoolean2);
}
}

Characters

The char data type stores a single character value that is surrounded by quotes ('').

public class MyNewClass {
public static void main(String[] args) {
char myChar = 'A';
System.out.println(myChar);
}
}

Strings

The String data type is used to store a sequence of characters surrounded by double quotes (""). A String in java is actually a non-primitive type(Description of non-primitive types in right sidebar), because a string is an object that has methods to perform operations on it. You will learn about objects in a later section.

public class MyNewClass {
public static void main(String[] args) {
String myString = "Welcome to the Java-Academy website";
System.out.println(myString);
}
}

Java Data Types

This page focuses on Java data types in more detail, concerning primitive & non-primitive data types with code examples.

Non-primitive data types are also known as reference types because they directly refer to an object. The main differences between a primitive and non-primitive data type are :

  • Primitive data types are already defined within java. Non primitive data types are created by the programmer with the exception of String.
  • Non-primitive data types can call methods to perform operations on the object whereas primitive types cannot.
  • A primitive data type must have a value whereas a non-primitive can be null.
  • A non-primitive type starts with a uppercase letter while a primitive type starts with a lower case letter.
  • Non primitive data types all have the same size while a primitive types size depends on its data type.
