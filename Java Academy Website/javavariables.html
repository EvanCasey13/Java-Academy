Java Variables

Variables in Java can be regarded as containers for storing certain data values, in Java there are many types of variables the five most used variables are :

  • String - This variable stores text surrounded by double quotes (" ") as seen in the MyNewClass class eg "This is my new class".
  • int - Stores whole integer values such as 12 or 20 (No decimal values).
  • char - Stores single characters surrounded by single quotes (' ') such as 'A' or 'B'.
  • boolean - Stores values with only two conditions either true or false.
  • float - This variable stores floating point numbers eg decimal numbers such as 12.10 or 20.5.

Java Variable Declarations (Creating Variables in Java)

To create a variable in java, you have to choose a type and give it a value :
Syntax : type variable = value;
Type is one of the five we have listed above eg (String or int), variable is a name you assign to the type you choose eg (y or name). The equal sign is used to assign a value to the variable. Java Variables

This page focuses on Java Variables which are used to store different data values and gives examples of these different types of variables applied in physical code.

Every variable in java must be identified with a unique name, this unique name is known as an identifier. Identifiers can be descriptive as in the examples to the left eg (myAge or firstName) or can be short such as x or y as a unique name. When creating a identifier it is now considered both good and common practice to have a identifier be descriptive in nature to allow for more understandable and supportable code.

Rules for creating variables
  • Names can contain letters, digits & underscores.
  • Names must begin with a letter.
  • Names are case-sensitive eg (firstName and firstname are two different variables).
  • Reserved words such as keywords eg (int or String) cannot be assigned as variable names.
  • Names must start with a lower case letter and cannot contain any spaces.
