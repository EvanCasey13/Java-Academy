Java Operators

Operators are very important in Java, they are used to perform operations on variables and values. There are many types of operators, the main operators we will focus on are Arithmetic, Assignment, Logical and Comparison operators

Arithmetic Operators

This is a list of every arithmetic operator used for mathematical calculations.

  • Addition Operator (+) This operator adds two variables together e.g. (x + y)
  • Substraction Operator (-) This operator subtracts one variable away from another variable e.g. (x - y)
  • Multiplication Operator (*) This operator multiplies one variable by another variable e.g. (x * y)
  • Division Operator (/) This operator divides one variables by another variable e.g. (x / y)
  • Modulus Operator (%) This operator return the remainder of a division between two variables e.g. (x % y)
  • Increment Operator (++) This operator increases the value of a variable by 1 e.g. (x++)
  • Decrement Operator (--) This operator decreases the value of a variable by 1 e.g. (x--)

Assignment Operators

This is a list of every assignment operator.

  • Operator : (=) e.g. (x = 5) is the same as (x = 5)
  • Operator : (+=) e.g. (x += 5) is the same as (x = x + 5)
  • Operator : (-=) e.g. (x -= 5) is the same as (x = x - 5)
  • Operator : (*=) e.g. (x *= 5) is the same as (x = x * 5)
  • Operator : (/=) e.g. (x /= 5) is the same as (x = x / 5)
  • Operator : (%=) e.g. (x %= 5) is the same as (x = x % 5)
  • Operator : (&=) e.g. (x &= 5) is the same as (x = x & 5)
  • Operator : (|=) e.g. (x |= 5) is the same as (x = x | 5)
  • Operator : (^=) e.g. (x ^= 5) is the same as (x = x ^ 5)
  • Operator : (>>=) e.g. (x >>= 5) is the same as (x = x >> 5)
  • Operator : (<<=) e.g. (x <<= 5) is the same as (x = x << 5)

Logical Operators

This is a list of logical operators which are used to figure out logic between two variables.

  • And Operator : (&&), this operator returns true if both statements are true e.g. (x < 3 && y > 6).
  • Or Operator : (||), this operator returns true if a single statement is true e.g. (x < 3 || y >6).
  • Not Operator : (!), this operator reverses a result from a statement, if true is the result then false will be returned e.g. !(x < 3 && y > 6).

Comparison Operators

This is a list of comparison operators which are used to compare two variables.

  • Equal to Operator : (==) e.g. (x == y) for example (3 == 5) which is false.
  • Not equal to Operator : (!=) e.g. (x != y) for example (3 != 5) which is true.
  • Greater than Operator : (>) e.g. (x > y) for example (3 > 5) which is false.
  • Less than Operator : (<) e.g. (x < y) for example (3 < 5) which is true.
  • Greater than or equal to Operator : (>=) e.g. (x >= y) for example (3 >= 5) which is false.
  • Less than or equal to Operator : (<=) e.g. (x <= y) for example (3 <= 5) which is true.
Code examples

public class MyNewClass {
public static void main(String[] args) {
int x = 3;
int y = 5;
System.out.println(x + y); //Will print 8 to the console (Arithmetic example)
System.out.println(x == y); //Will print false to the console (Comparison example)
System.out.println(x += 10); //Will print 13 to the console (Assignment example)
System.out.println(x > 2 && y < 7); //Will print true to the console (Logical example)
}
}
