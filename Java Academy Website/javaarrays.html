Java Arrays

Java Arrays are used to assign multiple values to a single variable instead of making a separate variables for each value, arrays are declared using square brackets ([]).

Below an array of String objects called cars has been declared.

String[] cars;

To add values to this array, place them in a comma-separated list with curly braces.

String[] cars = {"BMW", "OPEL", "VOLKSWAGEN"}; //An array of strings
int[] ages = {12, 21, 19, 8}; //An array of integers

Accessing elements within an array

Elements are the values within the array, each element can be accessed by using the index number. This code will access the first car in the array of cars.

Note arrays start at index 0 e.g. [0] is the first element with [1] being the second element

public class MyNewClass {
public static void main(String[] args) {
String[] cars = {"BMW", "OPEL", "VOLKSWAGEN"}; //An array of strings
System.out.println(cars[0]); //Prints BMW
}
}

Change an element within an array

You can change the value of a specific element in an array by using an index value. Check below for an example.

public class MyNewClass {
public static void main(String[] args) {
String[] cars = {"BMW", "OPEL", "VOLKSWAGEN"}; //An array of strings
cars[0] = "Mazda";
System.out.println(cars[0]); //Prints Mazda instead of BMW
}
}

Array length

Like a string, arrays are non-primitive and have methods that can be used on them, one we have seen before islength. This returns the amount of elements within an array. e.g.

public class MyNewClass {
public static void main(String[] args) {
String[] cars = {"BMW", "OPEL", "VOLKSWAGEN"}; //An array of strings
System.out.println(cars.length); //Prints amount of elements in the car array
}
}
